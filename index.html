Naar boven

Titel van sprint

Om mijn leerdoel te kunnen behalen heb ik een deskresearch uitgevoerd naar het leermateriaal dat ik kan gebruiken. Ik heb een selectie van woorden gekozen met betrekking tot het onderwerp: JavaScript, learn, material, course, book. Ik heb ook in het Nederlands en in het Engels gezocht. Er werd vaak vanuit experts het boek ‘Eloquent JavaScript’ gesuggereerd, omdat hij dingen in diepgang uitlegt. Ik vind een boek voor mij altijd beter, omdat ik dan wordt gedwongen om de code te typen en uitproberen. Dit doe ik echter niet bij het volgen van een video, omdat de code en het resultaat altijd worden vertoont door de maker van de video. Ik ben ook opzoek gegaan naar uitdagende projecten die ik met mijn eigen logica kon ontwikkelen. Ik heb een paar opties geselecteerd die de nieuwe onderwerpen van het boek kunnen bedekken. Uiteindelijk heb ik besloten om een password generator en het spel snakes and ladders te gaan bouwen. Voor het boek wilde ik een samenvatting opleveren, zodat andere geïnteresseerde studenten hem kunnen gebruiken, deze samenvatting kan ook een goed referentiepunt voor mij zijn.
