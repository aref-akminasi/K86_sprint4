Web Accessibility

Tijdens mijn stage kwam ik erachter dat we tijdens de studie nooit eerder echt aandacht hebben besteedt aan Web Accessibility. Daarom wilde ik dit blok gebruiken om daar meer over te leren, omdat ik het erg belangrijk en interessant vindt. Ik heb eerst zelf onderzoek gedaan door verschillende documenten te lezen en filmpjes te kijken. Maar om er nog dieper op in te kunnen gaan heb ik de cursus “Introduction to Web Accessibility” gevolgd. Deze cursus was online en werd onder andere gegeven door mensen met een beperking zelf. Hierdoor kon ik me erg goed inbeelden en heb ik vooral veel kunnen leren over waarom de richtlijnen zo noodzakelijk zijn en hoe ze zijn ontstaan. Ik heb vervolgens zelf een verslag gemaakt met een duidelijk overzicht van de belangrijkste richtlijnen en waar allemaal op gelet moet worden. Dit heb ik voor het design en voor het coderen uitgeschreven en daarnaast heb ik puntsgewijs tips beschreven. Dit heb ik niet alleen voor anderen geschreven, maar vooral voor mezelf. Ik vond het super interessant om hier meer over te leren en mee aan de slag te gaan. Ook heb ik echt ontdekt dat hier mijn interesse ligt. Klik hier om mijn onderzoek te bekijken

Accessible portfolio

Web Accessibility vond ik zo interessant dat ik er graag nog meer over wilde leren. Ook had ik een nieuwe portfolio-website nodig dus dit kwam goed samen. Ik heb er toen voor gekozen om alles wat ik in het vorige blok geleerd heb toe te passen door een dubbel A toegankelijk portfolio te ontwerpen voor mezelf. Om nog een extra uitdaging toe te voegen heb ik dit gedaan in Figma, wat voor mij een nieuw programma is. Van tevoren heb ik bepaald welke elementen ik allemaal op mijn website wilde hebben en die heb ik vervolgens vergeleken met de WCAG-richtlijnen om alvast te bepalen hoe ik mijn website goed toegankelijk kan maken. Daarnaast heb ik veel voorbeelden opgezocht en heb ik zelf bedacht wat er allemaal belangrijk is. Ik heb een aantal elementen toegevoegd die zorgen voor een goede accessibility, zoals het menu waar het lettertype en lettergrootte kan worden aangepast en de optie voor contrast van kleur op de website. Later ga ik de hele website coderen en zorgen dat het dan een volledig toegankelijk portfolio wordt. Ik vond dit een leuke opdracht en ik blijf het interessant vinden, dus ik ben ook zeker van plan om er mee door te gaan.
